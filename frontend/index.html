The AI Framework

Our system uses a Multi-Factor Weighting Engine. Unlike static calendars, StudyPulse AI evaluates Complexity, Urgency, Performance Gaps, and Student Behavioral Context (Sleep, Attendance, Motivation) to generate a realistic prediction.

Dataset Variables

  • V1 Behavioral Factors: Sleep hours and motivation levels are used to adjust recommended daily capacity.
  • V2 Urgency Index: Days until exam inversely affects the priority multiplier.
  • V3 Performance Delta: The gap between current score and target determines remedial allocation.

The Optimization Logic

Weight Calculation (Simplified)

Score_Mod = (Target - Current) * 0.2
Urg_Mod = (1 / Deadline) * 10
Base_Priority = (Complexity * 2) + Score_Mod + Urg_Mod
